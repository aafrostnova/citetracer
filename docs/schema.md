# Report Schema

Top-level fields:
- `report_version`
- `paper_id`
- `pipeline_type`
- `citations[]`
- `summary`
- `risk_score`
- `requires_human_review`
- `metadata`

Citation fields:
- `citation_id`
- `verdict`
- `confidence`
- `evidence_sources`
- `conflicts`
- `adjudication_reason`
- `matched_candidate`
- `needs_human_review`
- `extraction_quality`
